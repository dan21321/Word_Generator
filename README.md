Word Generator is a program to automatically generate documents from Excel data and a Word template. 
It supports placeholders, auto-numbering, and .docx output.

### Requirements
- Microsoft Word (for .doc to .docx conversion)
- .NET Framework 4.7.2

### Usage
1. Select an Excel file.
2. Select a Word template.
3. Map placeholders to Excel columns.
4. Choose output folder.
5. Click **Generate**.
